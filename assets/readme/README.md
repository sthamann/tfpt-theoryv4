## The hero + terminal demo (drawn from source, text-exact)

Generated by [`make_readme_assets.py`](make_readme_assets.py) (Pillow — no external SVG renderer
needed), so every number is exact:

| File | What it is |
|---|---|
| `00_hero.png` | The single share image: `ONE SEED → E8 hull → SM · α⁻¹ · gravity/cosmology → 23 predictions · 3 engines`. |

```bash
python3 assets/readme/make_readme_assets.py
```
